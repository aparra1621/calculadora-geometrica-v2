# 📐 Calculadora Geométrica V2

Aplicación de consola en Java que permite calcular áreas y perímetros de figuras geométricas.

## 🧠 Funcionalidades

- Soporta múltiples figuras (círculo, cuadrado, rectángulo, triángulo, pentágono).
- El usuario puede elegir entre calcular **área** o **perímetro**.
- Almacena los resultados en un arreglo.
- Permite realizar varias operaciones en una sola ejecución.
- Manejo de errores de entrada.

## 🚀 Cómo ejecutar

1. Compilar:

```bash
javac CalculadoraGeometricaV2.java
